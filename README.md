# 💎 AI Diamond Jangad Generator

A modern diamond inventory and Jangad management system built with HTML, CSS, JavaScript, and Excel export functionality.

## 🚀 Features

* Add Diamond Stock
* Search by Stock ID
* Inventory Dashboard
* Total Carat Calculation
* Total Value Calculation
* Delete Stock Records
* Excel Export (.xlsx)
* Local Storage Database
* Responsive Design
* Real-Time Inventory Updates

## 📊 Diamond Information

The system stores:

* Stock ID
* Availability Status
* Shape
* Carat Weight (CRT)
* Color
* Purity
* Cut
* Lab
* Certificate Number
* Per Carat Price
* Total Price

## 🖥️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* LocalStorage
* SheetJS (XLSX)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/sbhavy1212-star/ai-diamond-jangad-generator.git
```

Open the project folder:

```bash
cd ai-diamond-jangad-generator
```

Run:

```bash
index.html
```

Or use VS Code Live Server.
